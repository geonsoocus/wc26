# Changelog

## [0.10.0] - 2026-06-04

### Added
- 매치 상세 페이지 (/match) — WC26 리워드 배너, 참가자 프로필+국기
- 네이션스팩 확정보상 시 좌상단 국기 배지 표시
- 디버그 패널에 매치 상세 링크

### Removed
- 스토어 현금 결제 옵션 제거 (토큰 전용으로 변경)

## [0.9.0] - 2026-06-04

### Changed
- 친구 카드에서 심볼 제거, 국기만 유지
- 친구 카드 하단 좌/우 라운드 제거 (우상단 라운드만 유지)

## [0.8.0] - 2026-06-04

### Changed
- 콘페티 색상 통일: 레드(#FF4029), 블루(#1570FF), 옐로우(#FFBE1A), 그린(#E0FF47) 4색

## [0.7.0] - 2026-06-04

### Added
- POM 획득 풀스크린 화면 (Y축 회전 + 플래시 + 콘페티)
- 우승국 예측 슬롯 탭 → 보유 조끼 선택 바텀시트
- 미션 섹션 라운드 박스 디자인 적용

### Changed
- 우승국 예측에서 조끼 이미지 표시 (없으면 국기 이모지)
- 슬롯 라벨: 국가명/선택하기/잠김
- 출석체크 프로그레스바 제거
- 매치데이 미션 → 매치데이로 타이틀 변경

## [0.6.0] - 2026-06-04

### Added
- 팩 탭별 URL 라우팅 (/vest?tab=friends 등)
- 프로필 상단에 팩 아이콘 + 갯수 배지 (탭 클릭 시 팩 탭 이동)

### Changed
- 탭바에서 팩 탭 제거 (프로필 배지로 접근)
- 기본 프로필 브라질 → 네덜란드 변경

## [0.5.0] - 2026-06-04

### Added
- 출석체크 미션 (6/11~7/19, 39일간 매일 네이션스팩 리워드)
- 주간 캘린더 + 전체 캘린더 바텀시트 (6월/7월 탭 전환)

### Changed
- 탭 이름 챌린지 → 미션으로 변경

## [0.4.0] - 2026-06-04

### Added
- 스토어 탭 (크루삭스, 매치볼, 캡, 토트백, 핀 뱃지)
- 토큰 심볼 SVG 아이콘 (검은 원 + 녹색 번개)
- 프로필 카드에 토큰 잔액 배지 → 스토어 이동
- 시나리오별 토큰 잔액 (신규 0, 활성 1200, 초대 100)

## [0.3.0] - 2026-06-03

### Added

## [0.2.0] - 2026-06-03

### Added
