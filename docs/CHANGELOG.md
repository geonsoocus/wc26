# Changelog

## [0.18.0] - 2026-06-08

### Added
- 미션 섹션에 우승국 예측 항목 추가 (클릭 시 우승국 선택 모달)
- 리워드 수령 시 선물상자 🎁 플라잉 애니메이션 (리워드 노드 → 팩 뱃지)
- 프로필 생성 완료 후 친구 갤러리 표시 (공유 버튼)

### Changed
- 매치데이 노드 확대 (66x72), 글로시 효과 제거 → 플랫 단색 디자인
- 매치 라벨 "매치데이 1/2/3"으로 변경
- 미참가 매치 노드: 밝은 회색 배경 + 아웃라인
- 완료 매치 노드: 체크/아웃라인 두껍게 강조
- 리워드 노드: 플랫 초록색, 라벨을 받기/미리보기/완료로 표시
- 리워드 미수령 시 CTA "리워드 받기"로 전환 (초록색)
- 노드-라인 간 공백 제거, 연결선 길이 2배 확대
- 미션 리스트 흰색 박스/패딩 제거, 타이틀 크기 업, 간격 조정
- 출석체크 + 친구초대 미션 섹션 통합

## [0.14.0] - 2026-06-05

### Added
- 프로필 생성 인트로 화면 (플랩 월드 + 지구본 회전 + 샘플 카드 3장)
