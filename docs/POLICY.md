# WC26 이벤트 정책서

## 개요
PLAB Football x FIFA World Cup 2026 이벤트 프로토타입.
월드컵 기간(2026.06.11 ~ 07.19) 동안 진행되는 조끼 컬렉션 이벤트의 UI/UX 프로토타입.

## 이벤트 구조

### 조끼 컬렉션
- 48개국 (2026 FIFA 월드컵 본선 진출국) 스타일 PLAB 조끼를 수집
- 조끼는 네이션스팩 오픈을 통해 획득
- 획득한 조끼로 우승국 예측 가능 (최대 3슬롯)

### 팩 시스템

#### 네이션스팩 (초록)
- 오픈 시 랜덤 국가 조끼 1개 획득
- 확정 보상이 있는 경우 팩 좌상단에 해당 국기 표시
- 획득 경로: 출석체크, 매치 참여 리워드, 웰컴 팩

#### 리워드팩 (파랑)
- 오픈 시 아이템 획득 (할인 쿠폰, 토큰, 무료 참가권 등)
- 획득 경로: 매치 참여 리워드, 웰컴 팩

### 토큰 시스템
- 리워드팩 오픈으로 토큰 획득
- 토큰으로 스토어 굿즈 구매 가능
- 현금 결제 없음 (토큰 전용)

### 미션

#### 출석체크
- 월드컵 기간(6/11~7/19, 39일) 동안 매일 출석 가능
- 출석 시 네이션스팩 1개 리워드
- 주간 캘린더 + 전체 캘린더(바텀시트) 제공

#### 매치데이
- 매치 참여 마일스톤 (MATCHDAY 1, 2, 3)
- 각 매치데이 달성 시 팩 리워드 제공

#### 우승국 예측
- 보유 조끼 중 선택하여 우승국 예측
- 최대 3슬롯, 슬롯별 잠금/해제 상태
- 동일 국가 중복 선택 불가

### POM (Plaber of the Match)
- 매치 참여 후 POM 선정 시 풀스크린 카드 표시
- Y축 회전 애니메이션 + 플래시 + 콘페티
- POM 카드: 노란 상단 바(POM 텍스트) + 프로필 사진 + 국기 배지 + 파란 하단 바

### 스토어
- 토큰으로 실물 굿즈 구매
- 상품: 크루삭스(500), 매치볼(2000), 캡(1500), 토트백(1000), 핀 뱃지(300)
- 토큰 부족 시 구매 불가

### 프로필
- AI 생성 월드컵 국가대표 스타일 프로필 이미지
- 국기 탭 또는 프로필 이미지 탭으로 프로필 선택/변경
- 프로필 생성 쿼터 존재 (시나리오별 상이)

### 친구
- 카드/리스트 보기 토글
- 카드 뷰: 프로필 사진 + 국기 배지 (심볼 없음, 우상단만 라운드)
- 리스트 뷰: 매너 평점순 정렬
- 프로필 미생성 친구에게 알리기 기능
- 초대 배너 (케이스3: 초대받은 유저)

## 시나리오

### 케이스1: 신규 유저
- 이벤트를 처음 발견한 유저
- 프로필 미생성, 조끼 미보유
- 웰컴 팩 2개 (네이션스팩 + 리워드팩)
- 토큰: 0

### 케이스2: 참여중 유저
- 프로필 생성 완료, 매치 참여 중
- 조끼 5개 보유 (KOR, BRA, FRA, MEX, GER)
- 팩 4개 보유
- 토큰: 1,200
- 기본 프로필: 네덜란드

### 케이스3: 초대받은 유저
- 친구의 공유 링크로 접근
- 프로필 미생성
- 웰컴 팩 2개
- 토큰: 100

## 페이지 구조

### /vest — 메인 이벤트 페이지
- 프로필 카드 (프로필 사진, 스탯, 토큰/팩 배지)
- 탭: 미션, 친구, 컬렉션, 스토어
- 팩은 프로필 배지에서 접근
- 각 탭별 URL 라우팅 (/vest?tab=friends 등)

### /match — 매치 상세 페이지
- 기존 매치 상세 UI에 WC26 요소 추가
- WC26 리워드 배너 (다크 배경, 조끼 이미지)
- WC26 이벤트 라벨 (노란 배너)
- 참가자 프로필 사진 + 국기 배지
